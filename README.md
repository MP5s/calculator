# Calculator Service

Это проект калькулятора, который принимает арифметические выражения и вычисляет их с помощью агентов.

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone <url>
   cd calculator
2. Установите зависимости:

go mod tidy
3. Запустите сервер:

go run ./cmd/calc_service/...
